EPOLL SYSTEM v3.1 - README

About script:
EPOLL 3.1 - Script for creating unlimited number of polls. Every poll can have up to 99 answers. Description of features:
- No database used. All information is stored in 3 files.
- No limit for polls number.
- Poll's form style can be configured.
- All operations done using administration panel: adding polls, deletion and statistic viewing.
- Results are shown in rows, cam be gray or multi color.
- Correct calculation of percents.
- If any answer was voted by less then 1% of voters, result will be shown with hudredth part (Example : 0,05). - Multiple number of forms can be displayed at same web page.
- Poll form is inserted using include() function.
- Poll options and its style are configured on web page were poll is located ( before function include() ).

Install and Testing:
1) Install following all files into separate folder on your server and set required permissions:
- add.php (Default)
- admin.php (Default)
- conf.php (Default)
- func.php (Default)
- form.php (Default)
- data.dat (777)
- pollcfg.dat (777)
- password.dat.php (777)
Creating poll:
2.1) Open in browser file admin.php and enter password: 1 (it should be changed)
2.2) Choose "Create new poll".
2.3) Enter unique poll name (Example: poll), question and number of possible answers.
2.4) If everything was correctly done step 2 will be available. Fill all answers fiels and click "Create poll".
2.5) On successful creation of poll a message will be shown.
Adding poll to web page:
3.1) Open file, where poll is required and add following code, change values of variables to configure:
PHP код <?
//Begin Epoll 3.1
$pollname="poll";
$p_path="scripts/epoll/";
$p_submit="Голосовать";
$p_align="center";
$p_width="250";
$p_font="verdana";
$p_txtsize="8pt";
$p_bcolor="#000000";
$p_bgcolor="#E5E5E5";
$p_txtcolor="#000000";
$p_bwidth="1";
$p_days="";
$p_coltype="0";
$p_rwidth="500";
include($p_path."form.php");
//End Epoll 3.1
?>
$pollname - Unique polls name, from created polls
$p_path - Path to script's folder
$p_submit - Text for "submit" button
$p_align - Poll table alignement
$p_width - Table width
$p_font - Font type
$p_txtsize - Font size
$p_bcolor - Border color
$p_bgcolor - Background color
$p_txtcolor - Text color
$p_days - Time between re-voting in days, for prevent spamming only don't change this value, it will be set to 5 seconds.
$p_coltype - Type of colomn background: 1- different color per colomn, 0 - same color (#cccccc) (recomended)
$p_rwidth - Result table width

3.2) Now open previously modified file in browser, if you see polls form everything was done correctly.
3.3) Try to vote. After filling form press "Vote" button. A window with results will open. If you see a message: "Vote successful", script is working.

4) In order to add another poll, same code can be used, but poll's name must be changed.
